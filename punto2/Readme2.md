
# Punto 2 – AFD para identificadores

## Descripción
AFD que reconoce identificadores según la expresión regular:  
`[A-Za-z][A-Za-z0-9]*`  
(comienza con letra, seguido de cero o más letras o dígitos).

## Archivos
- `afd_identificador.py`: Clase AFD y programa principal.
- `prueba.txt`: Archivo con casos de prueba.

## Requisitos
- Python 3.8 o superior.

## Ejecución
```bash
python3 afd_identificador.py
